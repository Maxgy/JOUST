# JOUST
Joust is a game where two ostrich-riding opponents battle for victory by colliding at a higher point than the other.

### Dependencies
* Python >= 3.7.1
* pygame >= 1.9.4

### Controls
#### Player 1:
* flap: X
* left: Z
* right: C
#### Player 2:
* flap: Up arrow
* left: Left arrow
* right: Right arrow
